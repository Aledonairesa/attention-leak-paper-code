# Usage Instructions

Run the script `doit.sh` with two arguments:

1. A text file containing the list of target websites (e.g., `websites.txt`).  
2. A number indicating how many calls to make for each website.  

For example:

```bash
bash doit.sh websites.txt 10
```

This will execute 10 calls per website listed in `websites.txt`.
